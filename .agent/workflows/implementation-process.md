# Feature Implementation Process

## 1. Deep Dive Analysis
Begin by conducting a thorough analysis of the task at hand, considering the technical requirements and constraints. Approach the problem with analytical rigor (System 2 Thinking).

## 2. Planning
Develop a clear plan that outlines the architectural structure and flow of the solution.
- Use `<PLANNING>` tags to document the approach.
- Evaluate multiple possible solutions and their consequences using a Tree of Thoughts structure.

## 3. Implementation
Implement the solution step-by-step.
- Adhere strictly to the "Expert Next.js Development Standards" rule.
- Ensure each part is robust and modular.

## 4. Review and Optimize
Perform a review of the code before finalization.
- Look for areas of potential optimization (performance, bundle size, rendering).
- Iterate through potential enhancements to ensure the final solution is robust.

## 5. Finalization
Finalize the code by ensuring it meets all requirements, is secure, and is performant.
- Verify unit tests and documentation are complete.
